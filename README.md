## 📋 개요

Catxi Backend는 가톨릭대학교 캠퍼스 근처에서 학생들의 안전하고 경제적인 택시 합승을 지원하는 RESTful API 서버입니다. 실시간 채팅, 위치 공유, 매칭 시스템을 제공하여 학생들이 편리하게 택시를 공유할 수 있도록 돕습니다.
